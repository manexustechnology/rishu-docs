# Introduction to Rishu 

Welcome to **Rishu** - your gateway to seamless crypto investing, built with everyday users in mind.

Rishu is a mobile-first application that empowers users to effortlessly buy and manage cryptocurrencies using familiar payment methods like **Google Pay**, **Credit Cards**, and other local **on-ramp** services.

Our goal is to make crypto accessible to everyone - even users without prior blockchain experience. With Rishu, getting started with crypto is as easy as shopping online.
